How to use (v1)

  1. Import your utility's service-line list (CSV/XLSX). Only a line-ID column is required; include pwsid to pull in the model's system priority.
  2. Open the Worklist — lines are ordered by the model's system-level priority, pending first.
  3. Investigate each line and record the system-side and customer-side material, basis, method, date. The overall LCRR classification is computed automatically.
  4. Export the SDWIS rollup (state submission) and the model-feedback file (retrains/improves the model).

Honest scope: prediction is system-level, not per-line; v1 captures and exports, line-level prioritization and the rehabilitation module come later. No cloud; data stays in ./data/register.db.
